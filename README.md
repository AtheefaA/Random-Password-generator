# Random Password Generator (Java)

This project is a simple Java program that generates random passwords.
It creates secure passwords using a combination of uppercase letters,
lowercase letters, numbers, and special characters.

## Features
- Generates strong random passwords
- Includes letters, numbers, and symbols
- Simple and beginner-friendly Java code

## Technologies Used
- Java
- Console-based application

## How It Works
The program randomly selects characters from a predefined set
and combines them to generate a secure password.

## Example Output
Generated Password: A7@kL9#p
